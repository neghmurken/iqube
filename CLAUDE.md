# iqube

## Dependencies 
 - Go
 - Raylib

## Architecture

/assets/ -> contains all assets for the game (images, fonts, sprites, levels, etc)
/internal/ -> Game model, rendering
/main.go -> Main loop, entrypoints
/iqube -> binary
